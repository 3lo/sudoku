# Sudoku solver

Requirements:
 - OpenCV
 - PIL
 - pytesseract
 
Idea:
 - Go https://sudoku.com
 - Run script and press F
 - Script captures screen cap of sudoku board
 - Uses backtracking to solve sudoku puzzle
 - Produce image with all the numbers filled in
